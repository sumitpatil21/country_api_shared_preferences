Country API App
Country API App is a mobile application that provides a list of countries retrieved from a RESTful API. Users can view detailed information about each country and save their favorite countries for quick access.

Features
Country List: Displays a list of countries with their names and corresponding IDs.
Country Details: Detailed view of selected country information, including:
Region
Status
Population
Capital
Area
Independence status
UN membership
Alternate spellings
Favorites: Save and manage favorite countries for quick access.
